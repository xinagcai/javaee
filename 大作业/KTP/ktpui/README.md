### 目录结构
```
assets: 静态文件
common：通用js，工具js
components：项目通用组件
    common：所有通用组件
    content：与项目业务相关的通用组件
network：axios异步请求封装
router：路由
store：vuex
views：单页面组件
```

### 配置文件
```
vue.config.js: 配置文件，别名、跨域
```
